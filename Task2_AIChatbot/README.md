# Task 2: AI Chatbot (AlphaBot)

## Project Overview
This project is a Java-based **AI Chatbot** developed as part of the CodeAlpha Java Development Internship. The chatbot features a modern graphical user interface (GUI) built with **Java Swing** and utilizes a rule-based Natural Language Processing (NLP) approach to interact with users.

## Features
*   **Interactive GUI:** A clean, user-friendly interface with a scrollable chat area and custom styling.
*   **Instant Responses:** Uses a `HashMap` based training system to provide immediate answers to common queries.
*   **Simulated Processing:** Includes a slight delay in responses to mimic real-time human conversation.
*   **Java Knowledge Base:** Pre-trained to answer basic Java programming questions (e.g., JVM, data types, features of Java).

## Technologies Used
*   **Language:** Java (JDK 8 or higher)
*   **Library:** Java Swing (for GUI)
*   **Concepts:** Event Handling, Multithreading (Timer), Collection Framework (HashMap).

## How to Run
1.  **Clone the Repository:** 
    ```bash
    git clone https://github.com
    ```
2.  **Open in IDE:** Import the project into Eclipse or IntelliJ IDEA.
3.  **Run the Application:** Execute the `AIChatbot.java` file.
4.  **Interact:** Type questions like "What is Java?" or "Who are you?" into the input field and click **SEND**.

## Sample Queries to Try
- "Hi" / "Hello"
- "What is Java?"
- "JVM stands for"
- "Features of java"
- "Data types in java"
